# S3 Policies

## PrincipalOrgId 
- limits access to s3 bucket only from specific organization

```
{
    "Version": "2012-10-17",
    "Id": "OnlySpecificAccount",
    "Statement": [
        {
            "Sid": "OnlyDeveloperAccount",
            "Effect": "Allow",
            "Action": [
                "s3:PutObject",
                "s3:GetObject"
            ],
            "Resource": "arn:aws:s3:::development-code/*",
            "Condition": {
                "StringEquals": {
                    "aws:PrincipalOrgID": ["o-1234567890"]
                }
            }
        }
    ]
}
```
